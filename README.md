# restaurant-list
## 環境建置與需求 (prerequisites)
* Node.js
## 安裝與執行步驟 (Installation and Execution)
1. Clone the repository
```
git clone https://github.com/homtele/restaurant-list.git
```
2. Install dependencies
```
cd restaurant-list
npm install
```
3. Run the project
```
npm run start
```
Navigate to http://localhost:3000
## 功能描述 (Features)
* 點選餐廳檢視詳細資訊
* 依照名稱或類別搜尋餐廳
